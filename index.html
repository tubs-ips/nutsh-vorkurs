Die Nut Shell – Vorkurs Informatik an der TU Braunschweig

Die Nut Shell ist ein interaktives Tutorial, mit dem Sie den Umgang mit der bash-Shell und Kommadozeilenwerkzeugen erlernen können.

Installation der Nut Shell

  1. Kopieren Sie folgende Zeile in die Zwischenablage:
    source <(curl -s tubs-ips.github.io/nutsh-vorkurs/setup)
  2. Öffnen Sie ein Terminalfenster (auf den Unirechnern: Oben links Anwendungen → Diensprogramme → Terminal, in dem eine bash-Shell laufen sollte (dies ist Standard).
  3. Kopieren Sie die Zeile hinein und drücken Sie Enter. Die Installation startet.
  4. Im farbigen Menü, das sich öffnet, können Sie eine 1 eingeben und Enter drücken, um die erste Lektion zu starten.

Starten der Nut Shell

Nach erfolgreicher Installation können Sie die Nut Shell über das Kommando nutsh starten.

Systemvoraussetzungen

Sie können die Nut Shell an drei Orten installieren und benutzen:

  • Neu: Über SSH auf dem GITZ-Server (von überall aus)

    Hierbei behalten Sie Ihren aktuellen Fortschritt, Sie können allerdings keine grafischen Programme starten (gedit, acroread). Verbinden Sie sich per ssh (Linux, Mac OS) oder Putty (Windows) mit dem Server izcip01.ibr.cs.tu-bs.de

  • In den Linux-Räumen des GITZ
  • Auf Ihrem eigenen Rechner

Auf den Rechnern des Vorkurses und auf dem SSH-Server ist bereits alles eingerichtet, auf Ihrem eigenen Rechner brauchen Sie folgende Dinge:

  1. Eine Linux-Distribution, für Anfänger empfehlenswert sind Ubuntu oder openSUSE.
  2. Den Compiler der Programmiersprache "Go", mindestens in Version 1.1. Üblicherweise heißt das Softwarepaket golang.
  3. Git, das Paket heißt fast überall git.
  4. Zur Bearbeitung der Lektionen brauchen Sie verschiedene Programme, nämlich mindestens Java und pdflatex. Wenn Sie merken, dass Ihnen ein weiteres Programm fehlt, ist es kein Problem, es während der Lektion nachzuinstallieren.

Alternativ können Sie eine bereitgestellte Ubuntu-Mate VM benutzen:

  1. Installieren Sie eine passende Version von VirtualBox.
  2. Installieren Sie ebenfalls das VirtualBox 5.X.X Oracle VM VirtualBox Extension Pack, welches ebenfalls auf der Downloadseite angeboten wird. Klicken Sie hierzu auf All supported platforms.
  3. Laden Sie die VM herunter und importieren Sie diese mittels Doppelklick in VirtualBox.
  4. Das Passwort für die VM ist informatik.
  5. Die VM ist bereits passend eingerichtet.
