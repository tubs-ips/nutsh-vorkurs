Installation der Nut Shell

  1. Kopieren Sie folgende Zeile in die Zwischenablage:
    source <(curl -s nutsh.morr.cc/setup)
  2. Öffnen Sie ein Terminalfenster (zu finden unter Anwendungen → Systemwerkzeuge → Terminal)
  3. Kopieren Sie die Zeile hinein (mit Rechtsklick → Einfügen) und drücken Sie Enter. Die Installation startet.

Starten der Nut Shell

Nach erfolgreicher Installation können Sie die Nut Shell über das Kommando nutsh starten.
