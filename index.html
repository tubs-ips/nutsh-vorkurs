Vorkurs Informatik zum WS 13/14

Systemvoraussetzungen

Wenn Sie die Nut Shell während oder nach dem Vorkurs auf Ihrem eigenen Rechner installieren möchten, brauchen Sie folgende Dinge:

  1. Eine Linux-Distribution, für Anfänger empfehlenswert sind Ubuntu oder openSUSE.
  2. Den Compiler der Programmiersprache "Go". Üblicherweise heißt das Softwarepaket golang.
  3. Git, das Paket heißt fast überall git.

Auf den Rechnern des Vorkurses ist bereits alles eingerichtet.

Installation der Nut Shell

  1. Kopieren Sie folgende Zeile in die Zwischenablage:
    source <(curl -s nutsh.morr.cc/setup)
  2. Öffnen Sie ein Terminalfenster (zu finden unter Anwendungen → Systemwerkzeuge → Terminal)
  3. Kopieren Sie die Zeile hinein (mit Rechtsklick → Einfügen) und drücken Sie Enter. Die Installation startet.

Starten der Nut Shell

Nach erfolgreicher Installation können Sie die Nut Shell über das Kommando nutsh starten.
